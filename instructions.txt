This is a multi lingual application example with ReactJS
This project is created using Create React App.

Command: npx create-react-app language-example

Functionality:
---------------
Supports En and Ar languages
Language is persisted in cookie now. Later I will read from URL

When language is changed, I reload the browser to show language changes
I am not using React Context or global state to manage language change.
Language is persisted in Cookie.
